Everything listed here is a stable public API you can import from `agency_swarm`.

```python
from agency_swarm import Agency, Agent, BaseTool, function_tool
```

## Core Classes

| Export | Purpose | Docs |
|--------|---------|------|
| `Agency` | Multi-agent orchestration container | [API Reference](/references/api#agency-class) |
| `Agent` | Individual agent with tools and instructions | [API Reference](/references/api#agent-class) |
| `AgencyContext` | Shared context passed between agents | [Agency Context](/additional-features/agency-context) |
| `MasterContext` | Full execution context (agency + user data) | [Agency Context](/additional-features/agency-context) |
| `ThreadManager` | Manages conversation threads | [Deployment](/additional-features/deployment-to-production) |
| `PersistenceHooks` | Callbacks for saving/loading threads | [Deployment](/additional-features/deployment-to-production) |
