Welcome to **Agency Swarm**, an open-source framework for creating and orchestrating multi-agent AI systems.

<Info>
Agency Swarm makes it easy to build collaborative AI systems by providing tools for agent coordination, task management, and communication.
</Info>

## Key Features

<CardGroup cols={2}>
  <Card title="Modular Architecture" icon="cubes">
    Flexible configuration of AI agents built on OpenAI's Assistants API
  </Card>
  <Card title="Multi-Agent Collaboration" icon="users">
    Create swarms of specialized agents that work together to accomplish complex tasks
  </Card>
  <Card title="Easy Integration" icon="plug">
    Simple tools and APIs for coordinating multiple AI agents
  </Card>
  <Card title="Extensible Framework" icon="code">
    Customize agent roles, capabilities and interactions to fit your needs
  </Card>
</CardGroup>
