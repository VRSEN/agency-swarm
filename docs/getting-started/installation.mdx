Set up Agency Swarm in a few simple steps.

<Tip>
    Make sure you have `Python >=3.10 <=3.12` installed on your system before you proceed.
</Tip>

## Step-by-Step Installation

<Steps>

<Step title="1. Install via pip" icon="download" iconType="solid">

<CodeBlock language="bash">
pip install agency-swarm
</CodeBlock>
</Step>

<Step title="2. Verify the Installation" icon="check-circle" iconType="solid">

<CodeBlock language="bash">
python -c "import agency_swarm; print('Agency Swarm is installed successfully.')"
</CodeBlock>
</Step>

<Step title="3. Run a Demo" icon="play" iconType="solid">

<CodeBlock language="python">
from agency_swarm import Agent, Agency

agent = Agent(name="Assistant", role="Provide helpful responses.")

agency = Agency(agents=[agent])
agency.run_demo()
</CodeBlock>
</Step>

</Steps>

## Troubleshooting

<Tip>
If you encounter issues during installation, ensure that your Python and pip versions meet the requirements.
</Tip>

If problems persist, check the [FAQ](../faq/faq.md) or [submit an issue](https://github.com/VRSEN/agency-swarm/issues).
