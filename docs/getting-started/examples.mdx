# Examples

Explore practical examples and tutorials to get the most out of Agency Swarm.

## Agency Examples

Examples of agencies can be found in the [agency-swarm-lab](https://github.com/VRSEN/agency-swarm-lab) repository:

- [WebDevCrafters](https://github.com/VRSEN/agency-swarm-lab/tree/main/WebDevCrafters) - A web development agency that builds responsive web applications using Next.js, React, and MUI.
- [CodeGuardiansAgency](https://github.com/VRSEN/agency-swarm-lab/tree/main/CodeGuardiansAgency) - An agency that runs on the backend using GitHub actions and submits code reviews on pull requests, according to your SOPs.
