## Feature Overview

n8n is a flexible workflow automation platform that lets you connect dozens of services in visual flows. The Agencii **n8n community node** (`n8n-nodes-agencii`) connects your agents to these workflows so your agencies can receive tasks, run tools, and send back answers directly inside n8n.

<Tip>To protect your privacy and that of your clients, none of your chats, conversations, or messages are stored in our database. ðŸ”’</Tip>

## Step 3: Use the Agencii Node in a Workflow

The Agencii node exposes a single operation designed to send messages to your agency and receive structured responses.

<Steps>
  <Step title="Configure the Agencii node">
    1. Add an **Agencii** node to your workflow.
    
    ![Agencii Send message to agency node with parameters filled in](/images/integrations/n8n-integration/n8n-send-message-node-params.png)
    ![Agencii node details with available actions](/images/integrations/n8n-integration/n8n-node-details-panel.png)
    
    2. Select your **Agencii API** credentials.
    3. Under **Resource**, choose **Chat**, and under **Operation**, select **Send Message**.
    4. Configure the main fields:
       - **Integration ID** *(required)* â€“ paste the Integration ID from your Agencii n8n Platform integration.
       - **Message** *(required)* â€“ the prompt or instruction you want your default agent to process.
       - **Session ID** *(optional)* â€“ reuse the `sessionId` from a previous run to continue the same conversation; leave empty to start a new session.

    
  </Step>

  <Step title="Understand the node output">
    The Agencii node returns a JSON payload with (among others):

    - `text` / `response` â€“ the main answer from your agency.
    - `sessionId` â€“ the session identifier to reuse for multi-turn conversations.
    - `n8nIntegrationId` â€“ the platform integration that handled the request.

    You can pass these values into downstream nodes (e.g., email, CRM, webhooks) to build complete automations.
  </Step>
</Steps>
