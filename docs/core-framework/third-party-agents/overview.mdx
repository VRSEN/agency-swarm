Agency Swarm supports importing third-party agents that are not built as normal Agency Swarm Python agents.

Use this section when you want Agency Swarm to stay in charge of orchestration, while another runtime handles one specialized part of the work.

## When To Use Third-Party Agents

Use this path when:

- You already have a separate agent runtime
- You want Agency Swarm to delegate work to that runtime
- You do not want to rebuild that runtime as a normal Agency Swarm agent

## What Agency Swarm Still Handles

- Top-level orchestration
- Communication flows
- FastAPI or app integration
- Frontend event flow
