Use this tool when your agent needs to run Python code, analyze data, or test snippets with state kept between calls.

## Usage Example

```python
from agency_swarm import Agent, IPythonInterpreter

data_analyst = Agent(
    name="Data Analyst",
    description="Analyzes data using Python",
    instructions="./instructions.md",
    tools=[IPythonInterpreter],
)
```

<Note>
This tool requires the optional Jupyter dependencies.
</Note>
