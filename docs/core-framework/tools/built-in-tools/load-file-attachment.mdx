Use this tool when your agent needs to read local images or PDFs and pass them as tool outputs.

<Warning>
Only supports images and PDF files. For other file types, use the Agents SDK `FileSearchTool` or `CodeInterpreterTool`.
</Warning>

## Usage Example

```python
from agency_swarm import Agent, LoadFileAttachment

file_processor = Agent(
    name="File Processor",
    description="Processes and analyzes local files",
    instructions="./instructions.md",
    tools=[LoadFileAttachment],
)
```

<Note>
The agent can use absolute paths or paths relative to the working directory.
</Note>
