Agency Swarm includes built-in tools for common tasks. Use these when you want quick, reliable functionality without extra setup.

## Quick Overview

## IPythonInterpreter

Run Python code with state preserved across calls. See the full guide: [IPythonInterpreter](/core-framework/tools/built-in-tools/ipython-interpreter).

## PersistentShellTool

Run shell commands with state preserved across calls. See the full guide: [PersistentShellTool](/core-framework/tools/built-in-tools/persistent-shell-tool).

## LoadFileAttachment

Load local images and PDFs as tool outputs. See the full guide: [LoadFileAttachment](/core-framework/tools/built-in-tools/load-file-attachment).
