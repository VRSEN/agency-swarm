The [Agency Starter Template](https://github.com/agency-ai-solutions/agency-starter-template) gives you a working agency with real tools and examples.

## Recommended Flow (Use This Template)

1. Open the template repo and click **Use this template**.
2. Create your new repo.
3. Clone your repo and run it:

```bash
git clone https://github.com/<you>/<your-repo>.git my-agency
cd my-agency

python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

echo "OPENAI_API_KEY=sk-..." > .env

python agency.py
```

**You should see:** A terminal prompt where you can chat with the agency.

<Accordion title="Alternative: clone the template directly" defaultOpen={false}>
```bash
git clone https://github.com/agency-ai-solutions/agency-starter-template.git my-agency
```
</Accordion>
