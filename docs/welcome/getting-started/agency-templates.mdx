## Use a Template

<Steps>
  <Step title="Choose a template">
    Select the **Agency Starter Template** when you need the Agencii-compatible setup with Docker. Use **Agency Code** when you want to experiment with or extend that example repository.
  </Step>

  <Step title="Clone the repository">
    Clone your chosen template repository to your local machine.

    <CodeGroup>
    ```bash Agency Starter Template
    git clone https://github.com/agency-ai-solutions/agency-starter-template.git
    cd agency-starter-template
    ```

    ```bash Agency Code
    git clone https://github.com/VRSEN/Agency-Code.git
    cd Agency-Code
    ```
    </CodeGroup>
  </Step>

  <Step title="Install dependencies">
    Create a virtual environment and install dependencies (Python 3.12+ recommended, 3.13 preferred).

    <CodeGroup>
    ```bash macOS/Linux
    python -m venv venv
    source venv/bin/activate
    python -m pip install -r requirements.txt
    ```

    ```bash Windows
    python -m venv venv
    venv\Scripts\activate
    python -m pip install -r requirements.txt
    ```
    </CodeGroup>
  </Step>

  <Step title="Configure environment">
    Create a `.env` file in the project root with your keys. At minimum:

    ```
    OPENAI_API_KEY=sk-...
    ```
  </Step>

  <Step title="Run the demo">
    Start the terminal demo to verify the setup.

    ```bash
    python agency.py
    ```
  </Step>

  <Step title="Customize agents and tools">
    Modify agent instructions, add or edit tools, and adjust communication flows. Refer to [From Scratch](/welcome/getting-started/from-scratch) for deeper explanations and code structure.
  </Step>
</Steps>
