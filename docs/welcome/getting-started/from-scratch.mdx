<Note>
Use this guide when you need to build everything manually. Otherwise, start from the [Starter Template](/welcome/getting-started/starter-template).
</Note>

<Steps>
  <Step title="Set Your OpenAI Key">
    Set `OPENAI_API_KEY` in `.env` (loaded automatically on init).
    ```
    OPENAI_API_KEY=sk-...
    ```
  </Step>
  <Step title="Create Project Structure">
    Create a basic layout like this:
    ```
    my_agency/
    - agency.py
    - agency_manifesto.md
    - developer/
      - __init__.py
      - developer.py
      - instructions.md
      - tools/
      - files/
      - schemas/
    ```
    `tools/` holds custom tools, `files/` holds local files (and optional vector store uploads), and `schemas/` holds OpenAPI schemas.

    <Accordion title="Create this layout with the CLI" defaultOpen={false}>
      ```bash
      agency-swarm create-agent-template "Developer"
      ```
      This generates the same folder structure for a new agent.
    </Accordion>
  </Step>
  <Step title="Create Tools">
    Add a tool in `developer/tools/search_database.py`.
    ```python
    from agency_swarm import function_tool

    @function_tool
    def search_database(query: str) -> str:
        return f"Results for: {query}"
    ```
    <Accordion title="BaseTool class version" defaultOpen={false}>
      ```python
      from agency_swarm import BaseTool
      from pydantic import Field

      class SearchDatabase(BaseTool):
          query: str = Field(..., description="Search query")

          async def run(self) -> str:
              return f"Results for: {self.query}"
      ```
    </Accordion>
    See [Custom Tools](/core-framework/tools/custom-tools/step-by-step-guide) for more options.
  </Step>
  <Step title="Define Agent">
    In `developer/developer.py`:
    ```python
    from agency_swarm import Agent

    developer = Agent(
        name="Developer",
        instructions="./instructions.md",
        tools_folder="./tools",
        files_folder="./files",
        schemas_folder="./schemas",
    )
    ```
    <Accordion title="Prefer explicit tool imports?" defaultOpen={false}>
      ```python
      from agency_swarm import Agent
      from developer.tools.search_database import search_database

      developer = Agent(
          name="Developer",
          instructions="./instructions.md",
          tools=[search_database],
      )
      ```
    </Accordion>
  </Step>
  <Step title="Create Agency">
    In `agency.py`:
    ```python
    from agency_swarm import Agency
    from developer.developer import developer

    agency = Agency(
        developer,
        shared_instructions="./agency_manifesto.md",
    )
    ```
    <Accordion title="Multi-agent example with communication flows" defaultOpen={false}>
      ```python
      from agency_swarm import Agency
      from ceo.ceo import ceo
      from developer.developer import developer

      agency = Agency(
          ceo,
          communication_flows=[(ceo, developer)],
          shared_instructions="./agency_manifesto.md",
      )
      ```
      Flows are directional: `(ceo, developer)` means CEO can message Developer.
    </Accordion>
    See [Communication Flows](/core-framework/agencies/communication-flows) for multi-agent setups.
  </Step>
  <Step title="Run Demo">
    <Tabs>
      <Tab title="Terminal">
        ```python
        agency.terminal_demo()
        ```
      </Tab>
      <Tab title="Backend (Sync)">
        ```python
        result = agency.get_response_sync("Hello!")
        print(result.final_output)
        ```
      </Tab>
      <Tab title="Backend (Async)">
        ```python
        import asyncio

        async def main():
            result = await agency.get_response("Hello!")
            print(result.final_output)

        asyncio.run(main())
        ```
      </Tab>
    </Tabs>
    For a web UI demo, see `agency.copilot_demo()` in [Visualization](/core-framework/agencies/visualization).
  </Step>
</Steps>
